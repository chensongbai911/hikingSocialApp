## 🆘 常见问题快速解决

### 问题 1: 无法连接到 http://localhost:5173

```
解决:
1. 检查前端是否运行 (npm run dev)
2. 检查端口 5173 是否被占用
3. 尝试使用另一个浏览器
4. 强制刷新: Ctrl+Shift+R
```

### 问题 2: 登录失败

```
解决:
1. 确认后端正在运行
2. 确认账户信息正确 (user1@test.com)
3. 查看浏览器控制台 F12 是否有错误
4. 查看后端启动日志
```

### 问题 3: 页面显示不完整

```
解决:
1. 刷新页面
2. 清除浏览器缓存
3. 检查网络连接
4. 尝试另一个浏览器
```

### 问题 4: 数据显示为空

```
解决:
1. 刷新页面
2. 检查是否登录
3. 查看浏览器控制台错误
4. 检查后端日志
```
