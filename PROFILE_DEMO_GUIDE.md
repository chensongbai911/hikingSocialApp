# Profile 功能快速演示指南

## 🎯 访问路径

### 1. 个人资料页（Profile）

**URL**: http://localhost:5174/profile

**展示内容**：

- 圆形头像（带相机图标）
- 昵称"山间清风" + 男性图标 + 年龄 28 标签
- "关于我"个人简介卡片
- 5 个徒步偏好标签（teal 色）
- 生活相册 4 张照片 + 添加按钮
- Teal 色"编辑资料"按钮
- "隐私设置"入口（带盾牌图标）

**交互操作**：

- 点击返回按钮 → 返回上一页
- 点击右上角设置按钮 → 控制台输出（待实现）

- 点击"隐私设置"入口 → 跳转到 /privacy-settings
- 点击"添加照片"按钮 → 控制台输出（待实现）

---

### 2. 编辑资料页（EditProfile）

**URL**: http://localhost:5174/edit-profile

**展示内容**：

- 头像上传区域（点击相机图标上传）
- 昵称输入框（预填"山间清风"）
- 性别下拉框（选项：男/女/其他）
- 年龄数字输入框（预填 28）
- "关于我"文本域（32/100 字符计数器）
- 已选偏好标签（5 个，可删除）
- "添加偏好"按钮（展开推荐标签）
- 推荐标签区域（14 个标签）
- Teal 色"保存修改"按钮

**交互操作**：

- 点击头像/相机图标 → 控制台输出（待实现上传）
- 修改昵称/性别/年龄 → 实时更新表单数据
- 在文本域输入 → 字符计数器实时更新
- 点击标签上的"X"按钮 → 删除该偏好
- 点击"添加偏好"按钮 → 展开/收起推荐标签
- 点击推荐标签 → 添加到已选偏好
- 已选标签在推荐区域变灰且不可点击
- 点击"保存修改"按钮 → 返回 Profile 页

---

### 3. 隐私设置页（PrivacySettings）

**URL**: http://localhost:5174/privacy-settings

**展示内容**：

- Teal 色提示卡片（保护隐私说明）
- 4 个 Toggle 开关：
  1. 隐藏真实姓名（默认开启）
  2. 隐藏准确年龄（默认关闭）
  3. 仅好友可查看徒步记录（默认关闭）
  4. 向非好友隐藏位置（默认开启）
- 黑名单管理入口（显示数量"3"）

**交互操作**：

- 点击返回按钮 → 返回 Profile 页
- 切换 Toggle 开关 → 实时更新设置+控制台输出
- 点击黑名单管理 → 控制台输出（待实现）

## 🐛 常见问题

**Q1: 页面显示空白？**
A: 检查开发服务器是否运行：`npm run dev`

**Q2: 路由 404 错误？**
A: 确认 router/index.ts 已包含 /privacy-settings 路由

**Q3: 样式不生效？**
A: 确认 Tailwind CSS 已正确配置且服务器已重启

**Q4: Toggle 开关不工作？**
A: 检查浏览器是否支持 peer 伪类（需现代浏览器）

## 📊 数据格式

### Profile 数据

```typescript
{
  avatar: string,           // 头像URL
  nickname: string,         // 昵称
  gender: 'male' | 'female' | 'other',
  age: number,             // 年龄
  bio: string,             // 个人简介（max 100字符）
  preferences: string[],   // 偏好标签数组
  photos: string[]         // 相册照片URL数组（max 9张）
}
```

### PrivacySettings 数据

```typescript
{
  hideRealName: boolean,
  hideExactAge: boolean,
  friendsOnlyActivities: boolean,
  hideLocationFromNonFriends: boolean
}
```
