## 🐛 常见问题排查

### 问题 1：页面无法加载

**症状**：页面显示空白或错误
**排查**：

- [ ] 前端服务正在运行
- [ ] 路由配置正确
- [ ] 浏览器 console 无错误

### 问题 2：Tab 不能切换

**症状**：点击 Tab 无响应
**排查**：

- [ ] Vue 组件正确加载
- [ ] activeTab 数据绑定正确
- [ ] 点击事件触发

### 问题 3：颜色不正确

**症状**：按钮或标签颜色与设计不符
**排查**：

- [ ] Tailwind CSS 已加载
- [ ] 颜色类名正确
- [ ] 没有 CSS 覆盖

### 问题 4：图片不显示

**症状**：卡片封面显示为空或损坏图标
**排查**：

- [ ] 图片 URL 有效
- [ ] 网络连接正常
- [ ] 图片服务器可访问

### 问题 5：头像显示异常

**症状**：头像不是圆形或位置错误
**排查**：

- [ ] CSS 类名完整
- [ ] z-index 堆叠正确
- [ ] 浏览器缓存清除
