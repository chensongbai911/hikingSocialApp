## 🔧 技术栈

### 前端框架
- **Vue 3.4** - 渐进式 JavaScript 框架
- **TypeScript 5.3** - 类型安全语言
- **Tailwind CSS 3.4** - 实用 CSS 框架
- **Pinia 2.1** - 状态管理
- **Vite 5.4** - 快速构建工具

### 地图和位置服务
- **高德地图 API (AMap)**
  - `AMap.Geolocation` - 用户定位
  - `AMap.PlaceSearch` - 位置搜索
  - `AMap.Geocoder` - 地理编码

### 动画和样式
- **CSS Keyframes** - 流畅动画
- **Tailwind CSS** - 响应式设计
- **Safe Area** - 移动设备安全区适配
