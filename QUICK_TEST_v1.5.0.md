## 🚀 5分钟快速测试

### 1. 底部导航 (30秒)

访问 http://localhost:5174 登录后：

- [ ] 看到6个按钮：首页、发现、活动、消息、创建、资料
- [ ] 点击每个按钮有放大动画和底部圆点

### 2. 首页功能 (1分钟)

点击"首页"按钮（房子图标）：

- [ ] 显示欢迎横幅（头像+"嗨,用户名"）
- [ ] 有3个快速操作按钮
- [ ] 推荐活动可以横向滑动
- [ ] 推荐用户头像列表

### 3. 查看申请 (1分钟)

点击"活动"→"我发布的"：

- [ ] 招募中活动显示"X人申请中"
- [ ] 点击"查看申请→"跳转到申请列表
- [ ] 列表显示申请者信息

### 4. 审核功能 (1分钟)

在申请列表页：

- [ ] 待审核申请有"通过"和"拒绝"按钮
- [ ] 点击后弹出确认框
- [ ] 确认后状态更新，显示Toast提示

### 5. 编辑活动 (1分钟)

点击我发布的活动→活动详情：

- [ ] 底部有"编辑活动"按钮（仅组织者）
- [ ] 点击跳转到编辑页面
- [ ] 表单自动填充活动数据
