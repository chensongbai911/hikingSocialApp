## ✅ 核心改进

### 1. 底部导航栏重构

**改动前**:

- 5个按钮：发现、创建、消息、记录、资料
- 缺少首页入口
- 与PRD需求不符

**改动后**:

- **6个按钮**: 首页、发现、活动、消息、创建、资料
- 首页作为第一个入口，符合PRD设计
- 所有图标统一为 6x6 大小
- 优化内边距适配6按钮布局
- 保留创建按钮特殊样式（渐变+旋转动画）

**设计细节**:

```vue
<!-- 首页按钮（新增） -->
<router-link to="/">
  <svg class="w-6 h-6"><!-- 房子图标 --></svg>
  <div class="text-xs">首页</div>
</router-link>

<!-- 激活状态 -->
- 图标放大: scale-110 - 颜色: text-teal-500 - 底部圆点指示器
```

### 2. 路由配置优化

**改动前**:

```typescript
{ path: '/', redirect: '/discover' }
```

**改动后**:

```typescript
{
  path: '/',
  name: 'Home',
  component: () => import('@/components/pages/Home.vue'),
  meta: { requiresAuth: true },
}
```

**影响**:

- 登录后自动进入首页（不再是发现页）
- 首页成为应用默认落地页
- 符合"首页优先"的产品逻辑
