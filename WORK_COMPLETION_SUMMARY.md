## 💡 技术亮点

### 1. 大图沉浸式设计

```vue
<!-- 关键代码: 大图容器 -->
<div class="relative overflow-hidden rounded-2xl shadow-md">
  <img class="w-full h-48 object-cover" />
  <!-- 渐变遮罩 -->
  <div class="absolute inset-0 bg-gradient-to-t from-black/40"></div>
</div>
```

**优点**: 视觉冲击力强，让活动封面得到展示

### 2. 响应式筛选面板

```typescript
// 多维度状态管理
const activeFilters = ref({
  gender: [] as string[],
  age: [] as string[],
  level: [] as string[],
})

// 灵活的过滤逻辑
const filteredUsers = computed(() => {
  return users.value.filter((user) => {
    const genderMatch =
      activeFilters.value.gender.length === 0 || activeFilters.value.gender.includes(user.gender)
    // ... 其他维度类似
    return genderMatch && ageMatch && levelMatch
  })
})
```

**优点**: 支持多条件组合筛选，逻辑清晰

### 3. 日期时间格式化

```typescript
// 格式化函数
const formatDateTime = (dateTimeStr: string): string => {
  const date = new Date(dateTimeStr)
  const weekDays = ['日', '一', '二', '三', '四', '五', '六']
  const weekDay = weekDays[date.getDay()]
  return `${year}-${month}-${day} (星期${weekDay}) ${hours}:${minutes}`
}

// 输出示例
// "2024-01-20 (星期六) 06:00"
```

**优点**: 用户可快速了解活动日期和星期

### 4. 粘性头部导航

```vue
<!-- 粘性定位保持可见性 -->
<div class="sticky top-0 z-10 bg-white border-b">
  <!-- 页面标题 + Tab导航 -->
</div>
```

**优点**: 滚动时仍可切换 Tab，无需回到顶部

### 5. 友好的空状态

```vue
<!-- 无数据提示 -->
<div v-if="items.length === 0" class="text-center py-12">
  <div class="text-6xl mb-4">🥾</div>
  <p class="text-lg">还没有加入任何活动</p>
  <router-link class="...">去发现活动</router-link>
</div>
```

**优点**: 不是冷冰冰的提示，而是友好的引导
