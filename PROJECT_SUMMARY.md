## 🔧 故障排除

### 问题: 无法连接到前端

**解决**:

1. 检查 http://localhost:5173 是否可访问
2. 检查前端服务是否正在运行: `Get-Process node`
3. 检查端口是否被占用: `netstat -ano | findstr :5173`
4. 重启前端服务

### 问题: API 请求失败

**解决**:

1. 检查后端服务是否正在运行
2. 检查 CORS 配置是否正确
3. 检查数据库连接
4. 查看后端日志了解错误信息

### 问题: 数据库连接失败

**解决**:

1. 确保 MySQL 服务正在运行
2. 检查 MySQL 用户名密码是否正确
3. 检查数据库是否存在 (`hiking_app`)
4. 运行: `mysql -u root -p -e "CREATE DATABASE hiking_app;"`

### 问题: 样式显示不正确

**解决**:

1. 清除浏览器缓存 (Ctrl+Shift+Del)
2. 硬刷新页面 (Ctrl+Shift+R)
3. 清除 localStorage: `localStorage.clear()`
4. 重启前端服务
