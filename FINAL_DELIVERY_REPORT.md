## 🚀 立即开始

### 方案 1: 一键启动 (推荐)

**Windows**:

```powershell
D:\coze\start_app.bat
```

**Linux/Mac**:

```bash
chmod +x start_app.sh
./start_app.sh
```

### 方案 2: 手动启动

**启动后端** (Terminal 1):

```powershell
cd D:\coze\backend
npm run dev
```

**启动前端** (Terminal 2):

```powershell
cd D:\coze\frontend
npm run dev
```

### 访问应用
